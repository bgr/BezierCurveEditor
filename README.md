# BezierCurveEditor

This repo is a fork of [BezierCurveEditor by Arkham Interactive from Unity Asset Store](https://assetstore.unity.com/packages/tools/bezier-curve-editor-11278).

It differs from the version on the Asset Store - the first commit is the code by the original author, the rest are customizations made in the last couple of years we've been using the code. Those are mostly additions to existing functionality and shouldn't cause any breaking changes.
